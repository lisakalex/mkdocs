MkDocs + Material Theme
Docs: https://squidfunk.github.io/mkdocs-material/

python3 -m venv .venv
source .venv/bin/activate
pip install mkdocs-material
mkdocs new mydocs
cd mydocs
mkdocs serve
mkdocs build

To use GitHub Pages copy all the site dir contents to doc repo, commit and push.
MkDocs cannot be used directly because it requires a docs directory, which conflicts with GitHub Pages.
When GitHub Pages builds with Jekyll, it raises errors about the docs folder.
The issue may also be related to symlinks.
