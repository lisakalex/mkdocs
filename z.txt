Create a new virtual environment
uv venv --python 3.13

To install everything from your pyproject.toml:
uv sync

or from requirements.txt:
uv pip install -r requirements.txt

You can skip manual activation completely and use uv run:
uv run python source/manage.py runserver
uv pip install -r requirements.txt
uv pip install -r requirements-dev.txt

uv pip install python-decouple
python source/manage.py migrate
python source/manage.py runserver 0.0.0.0:8003

pkill -f mkdocs
